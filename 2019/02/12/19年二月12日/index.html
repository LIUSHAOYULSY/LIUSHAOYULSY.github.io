算法一：reduction

因为除法很慢，效率很低，所以将除法转化成乘法，位运算，和减法

求 x mod m

m的位数是k

使用条件: x位数不大于2*k。对同一个数反复取模。

使用方法（这里是非负数）：

计算常量 mu=b^2k / m

取模时：

q1 = x / b^(k-1)

q2 = q1 * mu

q3 = q2 / b^(k+1)r1 = x % b^(k+1)

r2 = (q3 * m) % b^(k+1)

r = r1 - r2

if ( r > m ) r -= m

return r
