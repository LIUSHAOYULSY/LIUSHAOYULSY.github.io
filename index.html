目前现阶段的学习计划大概是每天抽三小时理解《深入理解计算机系统原理》，两个小时学习C++，两个小时学习算法导论，研究一个算法

一：《深入理解计算机系统原理》简单的做下小笔记

1、shell：命令行的解释器，输出一个提示符，等待输入一个命令行，然后执行这个命令，若该命令行第一个单词不是一个内置的shell命令，就会假设是一个可执行文件的名字。

2、系统硬件组成：①总线②I/O设备（输入输出）每个I/O设备都通过一个控制器或设配器与I/O总线相连，功能都是在I/O总线和I/O设备之间传递信息③主存④处理器

3、软件包括：应用程序和操作系统

处理器，主存，I/O设备，硬件

4、文件是对I/O设备的抽象表示

虚拟内存是对主存和磁盘I/O设备的抽象表示

进程是对处理器、主存和I/O设8、备的抽象表示

5、操作系统内核管理进程间的转换

6、内核是系统管理全部进程所用的代码和数据结构的集合

7、堆：当调用malloc和free这样的C标准库函数时，堆可运行时动态地扩展收缩

栈：调用一个函数时栈会增长，从一个函数返回时栈会收缩

8、文件即字节序列

9、虚拟机包括操作系统、处理器、程序

10、计算机由处理器、存储器子系统组成

二：c++

1、感觉 类 在C++中是一个很神圣的东西。

2、个人觉得C++和C不同之处主要在于C++有OOP特性和泛型编程，这是C不具有的

#VPS搭建VPN

##设置账户

  1. 首先点击进入 Vultr官网 注册账号
    2.输入相关注册信息，然后点击创建账户
    ##搭建VPN
    1.第一步：选择机房位置 ，New York

2.Server Type这里 选择 Open VPN

3.然后直接点Deploy Now
服务器安装需要1分钟左右，安装好进入后台看到自己的账号密码

4.下载OpenVPN安装软件：单击图标　然后点击 Import – From Server

5.输入IP地址

6.输入用户名和密码 点击 Connect

##输入用户名和密码 点击 Connect
1.下载Xshell

2.开Xshell在新建会话窗口填写相关信息

3.点击用户身份验证，方法选择Password，填写服务器的用户名和密码

4.填写完之后点击确定。然后点击连接。

5.接受此主机密钥

6.界面提示：Please input password for shadowsocks-go，输入要设置的密码，然后按回车；

7.输入Shadowsocks的端口

##使用SHADOWSOCKS进行连接
1.填写信息

2..点击确定后，右击软件小图标，点启动系统代理
